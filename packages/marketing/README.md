# @website/marketing

Static scaffold of the TokenBuzz marketing landing page, ported from the Claude Design handoff.

## Dev

```bash
pnpm --filter @website/marketing dev
# or from the repo root:
pnpm dev:marketing
```

Runs at **http://localhost:3000**.

> This is a visual scaffold — no data fetching, auth, or API routes. All content is static placeholder copy from the design. Buttons and links are inert (`href="#"`).
